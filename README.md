# KIN SELECTION

**Kin Selection** (Noun): A form of natural selection. Some animals cooperate with relatives, even if this brings risk to themselves.

CAN YOU SURVIVE A DYSFUNCTIONAL FAMILY DINNER?

Can you de-escalate enough arguments to hold the family together? 

Once you've played through, you'll be assigned an animal! Post yours in the comments below (we want to see what is the most popular!!).


By **Eleanor Nguyen** (art, story, dialogue) and **Molly Stow** (programming)!


**Trigger warnings include:** miscarriage mention, snakes, cheating, Jordan Peterson, wayyyy more family drama than I thought it was possible for Eleanor to write in 3 days.
